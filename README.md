# dapp-boilerplate

## Tech Stack

- TypeScript, Next.js, Hardhat, Solidity, Tailwind CSS

## How to get started

1. please edit your environment variables

```bash
$ mv .env.sample .env
```

2. start frontend server

```
$ yarn
$ yarn dev
```

## How to deploy contract

